# CarRental
Web application to rent cars on specific date
# How to run
- Shift + F10
- mvn spring-boot:run
- mvn clean install and then java -jar target/CarRental-0.0.1-SNAPSHOT.jar
# Technology stack
- Java
- Spring Boot
- MySQL
- HTML Thymeleaf
# How to use app
Firstly we open homepage on http://localhost:8080/home
